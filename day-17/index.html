shape() function

CSS function for complex, responsive shapes.

Baseline 2026 · Newly available
Try the demo · Chrome for Developers · MDN · shape()

The new shape() function lets you clip an element to a complex, non-polygonal, responsive shape in CSS. This is a great option compared to clipping masks using clip-path: path(), and it works seamlessly with CSS custom properties to define coordinates and control points, making it more maintainable than SVG shapes. This also means you can animate your custom properties within shape() to create dynamic and interactive clipping.

Here's how to create a flag shape with curved top and bottom edges using shape(): 

.flag {
  clip-path: shape(from 0% 20px,
    curve to 100% 20px with 25% 0% / 75% 40px,
    vline to calc(100% - 20px),
    curve to 0% calc(100% - 20px)
      with 75% 100% / 25% calc(100% - 40px),
    close
  );
}

In this example, the horizontal coordinates use percentages to scale with the element's width, while the vertical coordinates for the curve's height use fixed pixel values, creating a responsive effect where the flag's wave remains constant regardless of the element's size.

Another example from the Chrome team uses a blob generator for shape() to create a fun frame effect — see the Chrome for Developers article linked above for that one.

Checking native support…

clip-path: shape() only recently reached Baseline "newly available" — some browsers in the wild may still lack it. Where it isn't supported, clip-path is simply ignored, so the box above renders as a plain rectangle and the slider has no visible effect.
