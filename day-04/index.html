::scroll-marker / ::scroll-button()

Carousel scroll affordances with native CSS pseudo-elements.

::scroll-button · 👍 23 · Limited availability
Scroll markers · 👍 19 · Limited availability
Try the demo · Chrome for Developers · MDN

This year, creating carousels and other scrolling experiences in CSS became much easier with the introduction of two new pseudo-elements: ::scroll-button() and ::scroll-marker(). These features let you create native, accessible, and performant carousels with just a few lines of CSS — no JavaScript required.

A carousel is essentially a scrollable area with added UI affordances for navigation: buttons to scroll back and forth, and markers to indicate the current position and allow direct navigation to a specific item.

Note: Chrome implements most of this feature, but has not yet implemented the prev and next values, hence why this is shown as not fully supporting the feature yet.

::scroll-button()

The ::scroll-button() pseudo-element creates browser-provided, stateful, and interactive scroll buttons. These buttons are generated on a scroll container and can be styled with CSS. They behave like regular <button> elements, are focusable, and are automatically disabled when scrolling is no longer possible in a given direction.

You can create buttons for any scroll direction — left, right, up, down — as well as logical directions like block-start and inline-end. When activated, a scroll button scrolls the container by approximately 85% of its visible area. 

.carousel::scroll-button(left) {
  content: "⬅" / "Scroll Left";
}

.carousel::scroll-button(right) {
  content: "⮕" / "Scroll Right";
}

Scroll markers

The ::scroll-marker pseudo-element represents a marker for an element within a scroll container. These markers are grouped in a ::scroll-marker-group and behave like anchor links, letting users jump directly to a specific item in the scroller — dot navigation for a carousel, or a table of contents for a long document.

Like ::scroll-button(), scroll markers are fully stylable with CSS: images, text, even counters. The :target-current pseudo-class styles the active marker that aligns with the currently-scrolled-to item. 

.carousel {
  scroll-marker-group: after;
}

.carousel > li::scroll-marker {
  content: ' ';
  width: 1em;
  height: 1em;
  border: 1px solid black;
  border-radius: 50%;
}

.carousel > li::scroll-marker:target-current {
  background: black;
}
