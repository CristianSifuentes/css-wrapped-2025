scroll-target-group

Turn a list of anchor links into connected scroll-markers.

👍 20 · Limited availability
Try the demo · MDN

In addition to ::scroll-button() and ::scroll-marker, CSS carousels include another neat feature: scroll-target-group. This designates an element as a container for a group of navigation items, like a table of contents — turning a manually-created list of anchor links into scroll-markers that can navigate the page.

Pair scroll-target-group with the :target-current pseudo-class to style the anchor whose target is currently visible. This gives you the power of ::scroll-marker from the CSS Carousel API, but with the flexibility of using your own HTML elements as markers — full control over their styling and content.

What you need for a scroll-spy nav

A list of anchor links
Each one points to a different section of the page (href="#section-1", etc.).
scroll-target-group: auto
Applied to the container of those links — no JavaScript scroll listeners required.
<nav class="toc">
  <ul>
    <li><a href="#section-1">Section 1</a></li>
    <li><a href="#section-2">Section 2</a></li>
    <li><a href="#section-3">Section 3</a></li>
  </ul>
</nav>

<main>
  <section id="section-1">...</section>
  <section id="section-2">...</section>
  <section id="section-3">...</section>
</main>

The following CSS creates the scroll-target-group, then styles the table of contents so the link for the section currently in view turns red and bold: 

.toc {
  scroll-target-group: auto;
}

.toc a:target-current {
  color: red;
  font-weight: bold;
}
