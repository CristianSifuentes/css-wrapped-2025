# 🎁 CSS Wrapped 2025

<div align="center">

**21 novedades. 21 días. 21 videos.**

Descubrí las 21 novedades más importantes de CSS en 2025, explicadas en videos cortos, prácticos y al grano.
A lo largo de 21 días 📅 exploramos una nueva feature cada día, con ejemplos claros y aplicación real en proyectos de desarrollo web moderno.

Ideal para frontend devs que quieren mantenerse actualizados 🚀 y dominar lo último que trae CSS.

</div>

---

## ✨ Sobre el proyecto

`CSS Wrapped 2025` es una serie de 21 mini-demos en HTML, CSS y JS puro, una por cada novedad de CSS.
Cada día vive en su propia carpeta (`day-01`, `day-02`, … `day-21`) con su propio `index.html`, `style.css` y `script.js`, para que cada feature se explore de forma aislada y reproducible.

```
_CSSWrapped2025/
├── index.html            # landing con el índice interactivo
├── css/style.css         # estilos compartidos del sitio
├── js/main.js            # script compartido del sitio
├── day-01/ … day-21/     # una carpeta por cada novedad de CSS
│   ├── index.html
│   ├── style.css
│   └── script.js
└── README.md
```

## 🚀 Cómo ver las demos localmente

```bash
git clone https://github.com/CristianSifuentes/css-wrapped-2025.git
cd css-wrapped-2025
open index.html   # o levantá un server estático, ej: npx serve .
```
